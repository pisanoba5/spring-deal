# 중거거래사이트 포트폴리오

## Introduce
- 중고거래사이트를 만들어봤습니다. 로그인 후 상품을 등록하고 찜하기 기능과 판매자와 원활한 거래를 돕는 1:1채팅 기능을 추가하였습니다.  
* 개발시간 : 21.02.15 ~ 21.03.21
* 개발환경
  - IDE : Eclipse IDE for java EE Developers
  - JAVA : JDK 1.8_251
  - Tomcat : Apache Tomcat 8.5
  - Maven : Maven 3.6.0
  - DB : Mysql
  - Framwork : spring
  - front-end : bootstrap
